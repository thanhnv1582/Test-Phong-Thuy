# Getting started

# YARN is recommend for this project!!!

## Clone repository:

### git clone https://github.com/BobbyAMK/DT_Fengshui.git

## Install package:

### yarn install --save

or

### npm install --save

## start project:

### yarn start and yarn android

### or

### npm start and npm android

## build:

### yarn build

### or

### npm build
